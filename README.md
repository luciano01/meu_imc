## For developers
> To clone locally
```
$ git clone git@github.com:luciano01/meu_imc.git
$ cd meu_imc
```
> Open the project in an editor, if you use VS Code, follow this
```
$ cd meu_imc
$ code .
```
> To run, follow this on terminal
```
$ flutter clean
$ flutter pub get
$ flutter run --release
```
